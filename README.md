# 汽车销售管理系统

## 1.系统说明

- 此系统是为了完成xauat信息系统课设而创建，使用vue+spring boot框架

- 适配的操作系统Windows11/10

## 2.操作说明

先运行 **`配置检查.exe`** 检查环境，再运行 **`启动系统.exe`** 启动服务，用完运行 **`关闭系统.exe`** 关闭。

启动后访问：

- http://localhost:3000 — 客户端
- http://localhost:5173 — 管理端
